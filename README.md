### Setup
If you want to try this application locally, follow the steps below:

1. Clone the project using git clone `git@github.com:FuadNabi/My-Portfolio
.git`.

2. Run `cd My-Portfolio`.

3. Run `npm install`.

4. Run `npm start`.
